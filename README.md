# Development
You can develop Rxrubi.

## enviroment
- swift 5.2.4
- Xcode 11.6

## Library
- Alamofire
- JGProgressHUD
- RxSwift
- RxCocoa
- RxBlocking
- RxKeyboard
- UIKit

## Configuration
- UI implementation: Storyboard
- Architecture: MVVM

## Setup
Clone the project.
```
$ git clone this Project.
$ cd Rxrubi
$ pod install
$ open rubi.xcworkspace
```

## Run make setup
After setup is complete, Workspace automatically opens in Xcode.
